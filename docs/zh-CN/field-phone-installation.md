# Telebirr 手机安装 - 简单逐步操作指南

从手机第一次显示的 **欢迎** 页面开始，按照本指南操作。

## 向主管索取以下资料

1. 获取 Wi-Fi 名称和密码。
2. 获取 AirDroid 二维码。
3. 获取 OpenClaw 设置代码或二维码。
4. 获取设备网关 URL。
5. 获取 Telebirr Agent 激活码。
6. 获取 SIM 1 的 ICCID。
7. 获取 SIM 2 的 ICCID。
8. 准备好 SIM 1 的 Telebirr PIN。
9. 准备好 SIM 2 的 Telebirr PIN。

<!-- pagebreak -->

## 1. 放入两张 SIM 卡

1. 长按电源键。
2. 点击 **关机**。
3. 取出 SIM 卡托。
4. 将第一张 Telebirr SIM 卡放入 **SIM 1**。
5. 将第二张 Telebirr SIM 卡放入 **SIM 2**。
6. 将 SIM 卡托装回手机。
7. 长按电源键开机。
8. 等待手机显示 **欢迎** 页面。

## 2. 开始设置 AirDroid

1. 停留在第一个 **欢迎** 页面。
2. 在空白区域快速点击六次。
3. 等待二维码扫描器打开。
4. 点击主管提供的 Wi-Fi 名称。
5. 输入 Wi-Fi 密码。
6. 点击 **连接**。
7. 将摄像头对准 AirDroid 二维码。
8. 等待二维码扫描完成。
9. 点击 **继续**。
10. 点击 **下一步**。
11. 点击 **接受**。
12. Android 请求权限时，点击 **允许**。
13. 等待 Android 主屏幕出现。

<!-- pagebreak -->

## 3. 完成 Biz Daemon 设置

1. 打开 **Biz Daemon**。
2. 点击第一个尚未启用的权限。
3. 点击 **允许**。
4. 返回 **Biz Daemon**。
5. 点击下一个尚未启用的权限。
6. 点击 **允许**。
7. 重复以上操作，直到所有基本权限都已启用。
8. 点击 **无障碍**。
9. 点击 **Biz Daemon**。
10. 打开 **使用 Biz Daemon**。
11. 点击 **允许**。
12. 返回 **Biz Daemon**。
13. 点击 **使用情况访问权限**。
14. 点击 **Biz Daemon**。
15. 打开 **允许查看使用情况**。
16. 返回 **Biz Daemon**。
17. 点击 **安装未知应用**。
18. 打开 **允许来自此来源的应用**。
19. 返回 **Biz Daemon**。
20. 点击 **通知**。
21. 打开 **允许通知**。
22. 返回 **Biz Daemon**。
23. 点击 **电池优化**。
24. 点击 **允许** 或 **不优化**。
25. 返回 **Biz Daemon**。
26. 点击 **继续**。

<!-- pagebreak -->

## 4. 等待两个应用自动安装

1. 保持手机连接 Wi-Fi。
2. 保持手机连接充电器。
3. 等待 **OpenClaw** 图标出现。
4. 等待 **Telebirr Device Agent** 图标出现。

## 5. 将手机语言改为英语

1. 打开 **设置**。
2. 点击 **系统**。
3. 点击 **语言和输入法**。
4. 点击 **语言**。
5. 点击 **添加语言**。
6. 点击 **English**。
7. 点击 **United Kingdom** 或 **United States**。
8. 将 **English** 移到列表最上方。
9. 返回 **设置**。
10. 点击 **日期和时间**。
11. 打开 **自动设置时间**。
12. 打开 **自动设置时区**。

## 6. 允许应用持续运行

1. 打开 **设置**。
2. 点击 **应用**。
3. 点击 **OpenClaw**。
4. 点击 **电池**。
5. 点击 **不受限制** 或 **无限制**。
6. 返回 **应用**。
7. 点击 **Telebirr Device Agent**。
8. 点击 **电池**。
9. 点击 **不受限制** 或 **无限制**。
10. 返回主屏幕。

<!-- pagebreak -->

## 7. 连接 OpenClaw

1. 打开 **OpenClaw**。
2. 点击 **Connect**。
3. 点击 **Setup Code**。
4. 扫描 OpenClaw 二维码，或输入主管提供的设置代码。
5. 点击 **Pair** 或 **Submit**。
6. 等待主管批准这台手机。
7. 等待屏幕显示 **Connected** 或 **Online**。
8. 返回主屏幕。

## 8. 为 Telebirr Device Agent 授权

1. 打开 **Telebirr Device Agent**。
2. Android 请求短信权限时，点击 **允许**。
3. Android 请求电话权限时，点击 **允许**。
4. Android 请求通知权限时，点击 **允许**。
5. 打开 **设置**。
6. 点击 **无障碍**。
7. 点击 **已下载的应用**、**已安装的服务** 或 **无障碍服务**。
8. 点击 **Telebirr deterministic USSD control**。
9. 打开 **使用服务**。
10. 点击 **允许**。
11. 返回 **Telebirr Device Agent**。

<!-- pagebreak -->

## 9. 激活 Telebirr Device Agent

1. 点击 **Device Gateway URL** 输入框。
2. 输入主管提供的设备网关 URL。
3. 点击 **One-time activation code** 输入框。
4. 输入主管提供的 Telebirr Agent 激活码。
5. 点击 **Activate**。
6. 等待屏幕显示 **Activated**。
7. 点击 **Confirm OpenClaw is paired**。
8. 等待进入下一个页面。

## 10. 输入 SIM 1 的 Telebirr PIN

1. 点击 **SIM ICCID**。
2. 输入主管提供的 SIM 1 ICCID。
3. 点击 **Telebirr PIN (local only)**。
4. 输入 SIM 1 的 Telebirr PIN。
5. 点击 **Confirm Telebirr PIN (local only)**。
6. 再次输入相同的 Telebirr PIN。
7. 点击 **Store local SIM PIN**。
8. 等待保存成功信息。

<!-- pagebreak -->

## 11. 输入 SIM 2 的 Telebirr PIN

1. 点击 **SIM ICCID**。
2. 删除 SIM 1 的 ICCID。
3. 输入主管提供的 SIM 2 ICCID。
4. 点击 **Telebirr PIN (local only)**。
5. 输入 SIM 2 的 Telebirr PIN。
6. 点击 **Confirm Telebirr PIN (local only)**。
7. 再次输入相同的 Telebirr PIN。
8. 点击 **Store local SIM PIN**。
9. 等待保存成功信息。

## 12. 完成安装

1. 连接充电器。
2. 保持 Wi-Fi 开启。
3. 打开 **Telebirr Device Agent**。
4. 向主管发送以下消息：`PHONE SETUP DONE`。
5. 保持手机开机并连接充电器。
